## Hosting Locally

1. Install jekyll
1. Navigate to the repo and fire off ``bundle init`` and ``bundle add jekyll``
1. Fire up the service ``bundle exec jekyll serve``
1. Your page is hosted at ``localhost:4000`` or some other port.
